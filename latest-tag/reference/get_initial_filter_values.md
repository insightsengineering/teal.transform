# Returns the initial values for the `vals` widget of a `filter_spec` object

Returns the initial values for the `vals` widget of a `filter_spec`
object

## Usage

``` r
get_initial_filter_values(filter, datasets)
```

## Arguments

- filter:

  (`filter_spec`) the filter generated by a call to
  [`filter_spec()`](https://insightsengineering.github.io/teal.transform/reference/filter_spec.md).

- datasets:

  (`named list`) a list of reactive `data.frame` type objects.

## Value

named `list` with two slots `choices` and `selected`.
