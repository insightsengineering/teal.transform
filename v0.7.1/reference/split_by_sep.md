# Split by separator (matched exactly)

Split by separator (matched exactly)

## Usage

``` r
split_by_sep(x, sep)
```

## Arguments

- x:

  (`character`) Character vector, each element of which is to be split.
  Other inputs, including a factor return themselves.

- sep:

  (`character`) separator to use for splitting.

## Value

List of character vectors split by `sep`. Self if `x` is not a
`character`.
